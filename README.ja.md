# 聴牌でGO!（Tenpai de GO!）

麻雀の聴牌トレーニングゲームです。

- デモ：https://akymakr.github.io/tenpai_de_go/

## ゲームモード

### カジュアル（CASUAL）
- 9問 + BOSSステージ
- 各問60秒、3ライフ制
- 不正解または時間切れでライフを1消費（続行/あきらめる を選択）
- 1〜9問の正解でBOSSステージ用の秒数を累積

### ストーリー（STORY）
- 難易度が自動で上がる：初級（1〜5）→ 中級（6〜10）→ 上級（11〜15）→ BOSS（16）
- 各問30秒、3ライフ制
- 不正解または時間切れでライフを1消費（続行/あきらめる を選択）
- 1〜15問の正解でBOSSステージ用の秒数を累積

### サバイバル（SURVIVAL）
- 無限問題
- 15秒からスタート、ライフなし
- 正解で時間回復：
  - 初級：+4秒
  - 中級：+8秒
  - 上級：+12秒

## BOSSステージのルール

- BOSS は「それまでに累積した秒数」だけで挑戦（問題の基本秒数60/30は追加しない）。
- ただし累積が少ない場合は下限を保証：
  - カジュアル：最低60秒
  - ストーリー：最低30秒

## 難易度

- 初級：最大3面待ち
- 中級：最大6面待ち
- 上級：最大9面待ち

## 遊び方

1. 言語を選択
2. ゲームモードを選択
3.（カジュアル/サバイバル）難易度を選択
4. 聴牌の待ち牌をすべて選択
5. 1枚以上選択すると「回答を確認」が押せる
6. 時間切れ時に1枚でも選択していれば自動提出扱い

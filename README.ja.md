# 聴牌でGO!（Tenpai de GO!）

麻雀の聴牌トレーニングゲームです。

- デモ：https://akymakr.github.io/tenpai_de_go/

## ゲームモード

### カジュアル（CASUAL）
- 9問 + BOSSステージ
- 各問45秒、ライフ3
- 不正解または時間切れでライフを1消費（続行/あきらめる を選択）
- 1〜9問の正解で BOSS 用の秒数を累積

### ストーリー（STORY）
- 難易度が自動で上がる：初級（1〜3）→ 中級（4〜6）→ 上級（7〜9）→ BOSS（10）
- 各問30秒、ライフ3

### サバイバル（SURVIVAL）
- 無限問題
- 開始60秒、ライフなし
- 正解で時間回復：
  - 初級：+5秒
  - 中級：+10秒
  - 上級：+15秒

## BOSSステージのルール

- BOSS は「それまでに累積した秒数」だけで挑戦（問題の基本秒数 45/30 は追加しない）。
- ただし累積が少ない場合は下限を保証：
  - カジュアル：最低45秒
  - ストーリー：最低30秒

## 難易度

- 初級：最大3面待ち
- 中級：最大6面待ち
- 上級：最大9面待ち

## 遊び方

1. 言語を選択
2. ゲームモードを選択
3.（カジュアル/サバイバル）難易度を選択
4. 聴牌の待ち牌をすべて選択
5. 1枚以上選択すると「回答を確認」が押せる
6. 時間切れ時に1枚でも選択していれば自動提出扱い
